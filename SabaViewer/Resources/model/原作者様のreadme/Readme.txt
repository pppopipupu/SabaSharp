～モデル使用上のアドバイス～

MMD用モデルです。Toon調にエフェクトを設定したモデルも同梱します。

試験的に、「顔の基本的な動きをすべてボーンで実現する」ということをやってます。モーフもほとんどが
ボーンモーフです。手動操作もできるので、ちょっとした変化をつけるときにどうぞ。

各種拡大モーフは、ウエイトが若干狂う可能性があるのでご注意ください。

「靴消し」「素足」モーフを使用する場合、若干地面から浮きます。
